# plink

嵌入式实时波形可视化工具，专为 嵌入式 调试设计。通过 pyOCD 直连 DAPLink/STLink/JLink 探针，以最高 5000Hz 采样率实时读取 MCU 内存变量并绘制成示波器风格波形。

## 功能概览

### 核心采样

| 功能 | 说明 |
|------|------|
| 高速采样 | 纯上位机轮询模式，32 位对齐聚合读取引擎，实测 可达400Hz |
| ELF 符号解析 | 基于 pyelftools 静态解析，支持基本类型、结构体、枚举、指针链 |
| 多通道同步 | 同时监控多个变量，共享统一时间轴 |
| 环形缓冲区 | 高效 numpy 环形缓冲，长时间采样不丢帧 |

### 波形分析工具

#### 双游标与 ΔT/ΔV 差值测量

精准读取任意两个时刻的数值和差值，用于测量信号周期、幅值、延迟等参数。

- **Cursor A**（蓝色虚线）/ **Cursor B**（红色虚线），可独立放置和拖拽
- 底部信息栏实时显示 `游标A: t=xxx s  V=xxx`、`游标B`、`ΔT=xxx s  ΔV=xxx`
- 游标在视图滚动/缩放后保持绝对时间位置不变

#### 独立通道缩放与偏移 (Scale / Offset)

解决不同量级变量（如 3.3V 电压 vs 5000 RPM 转速）无法同时看清的问题。

- 监控表中每个通道可独立设置 `Scale`（乘法因子）和 `Offset`（加法偏移）
- 公式：`显示值 = (原始值 + Offset) × Scale`
- 各通道调节互不影响

#### 软件边沿触发器

让波形稳定显示，像真实示波器一样"触发同步"。

- **触发源**：任意已监控变量
- **触发边沿**：上升沿 / 下降沿
- **触发模式**：
  - `Normal`（连续）— 周期性信号自动重触发，波形稳定不漂移
  - `Single`（单次）— 首次触发后锁定波形，适合抓偶发异常（毛刺、过冲）
- 触发后视图以触发点为基准对齐，支持预触发回看

#### 局部统计 (Max / Min / Avg)

监控表中实时显示当前可见时间窗口内各通道的最大值、最小值、平均值，每 30ms 自动刷新。

#### 数据导出 CSV

工具栏 `[导出CSV]` 按钮，将波形数据导出为 CSV 文件，支持当前可见范围或全部数据。可被 Excel / Matlab 直接解析。

## 快捷键

| 按键 | 功能 |
|------|------|
| `A` | 在鼠标位置放置 Cursor A |
| `B` | 在鼠标位置放置 Cursor B |
| `↑` / `↓` | 切换游标观测的变量 |
| `Esc` | 清除所有游标 / 触发复位 |
| 鼠标左键拖拽 | 拖拽游标微调位置 |

| 滚轮 | 时间轴平移 |

## 安装

```bash
# 克隆仓库
git clone 
cd plink

# 创建虚拟环境
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Linux/macOS

# 安装依赖
pip install -r requirements.txt
```

### 依赖

- Python 3.10+
- PyQt6
- pyqtgraph

- numpy

## 使用

```bash
python -m src.main
```

### 基本流程

1. **加载 ELF 文件** — 选择编译生成的 `.axf` / `.elf` 文件，自动解析全局变量
2. **连接探针** — 选择调试探针（DAPLink / STLink / JLink），自动识别目标芯片
3. **添加变量** — 从变量树中双击或拖拽变量到监控列表
4. **开始采样** — 点击 **开始采样** 按钮，波形实时绘制
5. **分析** — 使用游标、触发器、缩放等工具分析波形

### 连接异常处理

| 异常类型 | 处理方式 | UI 反馈 |
|---------|---------|--------|
| USB 断开 | 停止采样，弹窗提示 | 🔴 状态栏变红 |
| SWD 通信超时 | 自动重试（指数退避） | 🟡 状态栏提示重试中 |
| 目标复位 | 自动重连 + 重新 attach | 🟡 状态栏提示重连中 |

断开后历史波形数据保留，重连后新数据追加到现有缓冲区。

## 项目结构

```
src/
├── main.py                    # 应用入口
├── sampling_engine.py         # 采样引擎
├── ring_buffer.py             # numpy 环形缓冲区
├── core/
│   ├── pyocd_backend.py       # pyOCD 调试后端
│   ├── elf_reader.py          # ELF 符号解析器
│   ├── symbol_cache.py        # 符号缓存
│   └── data_types.py          # 数据类型定义
├── ui/
│   ├── main_window.py         # 主窗口
│   ├── connection_panel.py    # 连接配置面板
│   ├── variable_list.py       # 变量树与监控列表
│   └── waveform_view.py       # 波形视图
└── engine/
    └── __init__.py
```
